BANGKIT Starter Project
======================

Struktur:
- backend/   : Laravel stubs (controllers, routes, migration). Copy into a Laravel project.
- frontend/  : React + Vite + Tailwind starter (run npm install inside frontend/)

Quick start (frontend):
1. cd bangkit/frontend
2. npm install
3. npm run dev

Backend:
- Copy backend files into a Laravel project or create a new Laravel project and replace routes/controllers.
- Configure .env and run migrations.

This starter provides pixel-perfect Tailwind utilities and sample pages (Landing, Login, Dashboard).
